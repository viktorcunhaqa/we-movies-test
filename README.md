# we-movies-test
Teste Wemovie - 3 Cenários e2e (TypeScript + Playwright)
Este repositório contém testes end-to-end (E2E) para o site [WeMovies QA](http://wemovies-qa.s3-website.us-east-2.amazonaws.com/), utilizando **Playwright**.
